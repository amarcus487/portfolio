Fulfillment Center Dashboard

During my last internship, I was able to cohesively create a hypothetical dashboard for any fulfillment center in Figma. I experienced both the design and prototyping behind this project to combine aesthetic with user-friendly features.

This tool combines both managerial and associate aspects to allow for an effective dashboard that is time efficient. The video provides a step-by-step tutorial of how my dashboard would work and what kind of information it would provide.
